## Serverless Vector Database

**Problem:** Milvus costs were $3,000/month for similarity search we only used a few hours per day.

**Solution:** Built an on-demand vector search solution using FAISS, Apache Arrow, and Kubernetes.

**Result:** Reduced costs to $200/month (93% reduction) while maintaining <50ms p99 latency.
